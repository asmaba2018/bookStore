Q1 - input element used to submit or send data to backend with type "submit" <input type="submit">.
Q2 - h1 tag used once in the document with main header <h1>.
Q3 - With sub-headers you can use h2 or h3 tags <h2> or <h3>.
Q4 - plain text could be included in p or paragraph tag <p>.
Q5 - document language should be declared in html tag with lang attribute <html lang="en">
Q6 - for a document reader it's better to use "10 to 20" as it would sound as what intended.
Q7 - Any div element could be accessed with tabs by adding tabindex attribute to it <div tabindex="0">.
Q8 - in web-development "ARIA" stands for "Accessible Rich Internet Applications".
